# Day 12 - Interface Configuration

## Overview
This Packet Tracer lab focused on configuring and verifying basic network interfaces on Cisco routers and switches.

## Skills Practiced
- Configured device hostnames
- Assigned IPv4 addresses to router and PCs
- Configured interface speed and duplex
- Added interface descriptions
- Disabled unused switch interfaces
- Verified configurations using Cisco IOS show commands
