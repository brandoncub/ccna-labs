# CCNA Day 14 - Static Routing

## Overview

This lab focuses on configuring and verifying IPv4 static routes between multiple routers using Cisco IOS.

## Topics Covered

- IPv4 Static Routing
- Next-Hop IP Configuration
- Directly Connected Networks
- Routing Table Verification
- End-to-End Connectivity Testing

## What I Configured

- Assigned IPv4 addresses to router interfaces
- Configured IP addressing on end devices
- Configured static routes using next-hop IP addresses
- Verified routing tables using `show ip route`
- Tested end-to-end connectivity with `ping`

## Skills Practiced

- Static route configuration
- Router CLI navigation
- IP addressing
- Routing verification
- Basic troubleshooting
