# Day 17 - VLAN Configuration

## Overview

Completed a VLAN configuration lab in Cisco Packet Tracer focused on subnetting, VLAN creation, switch configuration, and router interface setup.

## What I Practiced

- Created and named multiple VLANs
- Configured switch access ports
- Assigned devices to the correct VLANs
- Configured router interfaces as default gateways
- Applied /26 subnetting
- Verified connectivity with ping
- Observed ARP behavior during initial communication

## Notes

This lab reinforced how VLANs separate broadcast domains while allowing communication through a router. I also practiced calculating subnet ranges, assigning gateway addresses, and troubleshooting connectivity issues between hosts.
